# Theme DNA Reference

> A curated library of design DNA strands for the Art Director to mix and match.
> Read this for inspiration when creating unique per-course themes.
> Each "strand" is a complete recipe that can be combined with strands from other categories.

## Color DNA: 5 Premium Palette Recipes

### 1. Dark Premium (Science, Finance, Tech)
```
Background:   #0a1628  (deep navy-black)
Surface:      #1a2942  (elevated navy)
Surface-Alt:  #243650  (lighter navy)
Primary:      #635bff  (electric indigo)
Accent:       #00d4aa  (mint green)
Text:         #e2e8f0  (cool white)
Text-Muted:   #94a3b8  (slate gray)
Success:      #34d399  (emerald)
Error:        #f87171  (soft red)
Warning:      #fbbf24  (amber)
Border:       rgba(255,255,255,0.08)
```
**Personality:** Authoritative, sophisticated, data-driven. Like Stripe meets Bloomberg Terminal.
**Grain opacity:** 0.06 | **Glass style:** glass-dark | **Shadow tint:** indigo

**WHY this palette works:** The deep navy-black (#0a1628) creates a "cinema screen" effect —
content floats in darkness, making every color pop. Electric indigo (#635bff) as primary
ensures clickable elements are instantly recognizable without competing with content.
Mint green (#00d4aa) as accent provides the "reward" color for success states and CTAs.

## Font DNA: 10 Distinctive Pairings

### 1. Sharp Modern
- **Display:** Clash Display (800) -- geometric, futuristic, confident
- **Body:** DM Sans (400, 500) -- clean, readable, modern
- **Letter-spacing:** -0.03em headings, 0 body
- **Best for:** Tech, SaaS, innovation courses

**WHY this pairing works:** Clash Display's sharp geometry signals "cutting edge" while
DM Sans provides comfortable reading. The contrast between display and body weights
creates strong visual hierarchy — learners instantly see headings vs content.

### 2. Professional Trust
- **Display:** Gilroy (700) -- rounded geometric, approachable authority
- **Body:** Source Sans 3 (400, 600) -- government-grade readability
- **Letter-spacing:** -0.02em headings, 0.01em body
- **Best for:** Corporate, compliance, finance

### 3. Friendly Explorer
- **Display:** Space Grotesk (600, 700) -- quirky geometric, personality
- **Body:** DM Sans (400, 500) -- pairs perfectly, comfortable reading
- **Letter-spacing:** -0.02em headings, 0 body
- **Best for:** Science, exploration, onboarding

### 4. Elegant Editorial
- **Display:** Playfair Display (700, 900) -- classic serif, dramatic
- **Body:** Lora (400, 500) -- refined serif, literary
- **Letter-spacing:** -0.01em headings, 0.02em body
- **Best for:** History, literature, arts, leadership

### 5. Code Academy
- **Display:** JetBrains Mono (700) -- monospace with personality
- **Body:** IBM Plex Sans (400, 500) -- technical yet readable
- **Letter-spacing:** -0.01em headings, 0 body
- **Best for:** Programming, IT, data science

### 6. Warm Storyteller
- **Display:** Fraunces (700, 900) -- variable font with warmth
- **Body:** Source Serif 4 (400, 600) -- readable serif, trustworthy
- **Letter-spacing:** -0.02em headings, 0.01em body
- **Best for:** Social impact, humanities, coaching

### 7. Bold Energy
- **Display:** Plus Jakarta Sans (700, 800) -- geometric, energetic
- **Body:** Nunito (400, 600) -- rounded, friendly
- **Letter-spacing:** -0.03em headings, 0 body
- **Best for:** Marketing, sales, startup courses

### 8. Minimalist Focus
- **Display:** Outfit (600, 700) -- clean geometric, no-nonsense
- **Body:** Karla (400, 500) -- honest, practical
- **Letter-spacing:** 0.02em headings, 0.01em body
- **Best for:** Productivity, wellness, methodology

### 9. Arabic Modern
- **Display:** Noto Kufi Arabic (700) -- geometric Kufic, strong
- **Body:** Cairo (400, 600) -- excellent Arabic readability
- **Line-height:** 1.8 for body, 1.3 for headings
- **Best for:** Arabic-primary courses, NELC compliant

### 10. Arabic Elegant
- **Display:** Aref Ruqaa (700) -- calligraphic, traditional
- **Body:** Noto Sans Arabic (400, 500) -- clean, modern
- **Line-height:** 1.8 for body, 1.4 for headings
- **Best for:** Islamic studies, literature, cultural courses

## Background DNA: 6 Pattern Recipes

### 1. Film Grain
```css
.has-grain::before {
  content: '';
  position: absolute;
  inset: 0;
  opacity: var(--grain-opacity, 0.08);
  mix-blend-mode: overlay;
  pointer-events: none;
  background: url("data:image/svg+xml,...feTurbulence...");
}
```
**Effect:** Warm, cinematic texture. Works on every theme.
**When to use:** Always. This is the baseline texture.

### 2. Glassmorphism
```css
.glass {
  backdrop-filter: blur(12px);
  background: rgba(255,255,255,0.06);
  border: 1px solid rgba(255,255,255,0.08);
  border-radius: 16px;
}
```
**Effect:** Frosted glass floating over gradient backgrounds.
**When to use:** Dark themes, overlays, feature cards. Max 2-3 per viewport.

### 3. Mesh Gradient
```css
.mesh-gradient {
  background:
    radial-gradient(at 20% 30%, var(--color-primary) 0%, transparent 60%),
    radial-gradient(at 80% 70%, var(--color-accent) 0%, transparent 60%),
    radial-gradient(at 50% 50%, var(--color-secondary) 0%, transparent 80%);
}
```
**Effect:** Organic, flowing color blobs like Stripe's homepage.
**When to use:** Hero sections, intro slides, background layers.

### 4. Animated Gradient
```css
.animated-bg {
  background: linear-gradient(135deg, var(--bg-1), var(--bg-2), var(--bg-3));
  background-size: 400% 400%;
  animation: bg-shift 15s ease infinite;
}
```
**Effect:** Slowly shifting colors that feel alive.
**When to use:** Course intro, celebration screens, hero sections.

### 5. Starfield (Dark Themes)
```css
.starfield {
  background-image:
    radial-gradient(1px 1px at 20% 30%, rgba(255,255,255,0.8), transparent),
    radial-gradient(1px 1px at 40% 70%, rgba(255,255,255,0.5), transparent),
    radial-gradient(1px 1px at 60% 10%, rgba(255,255,255,0.6), transparent);
}
```
**Effect:** Night sky with twinkling stars. Pure CSS, no images.
**When to use:** Space/science themes, dark themes, astronomy courses.

### 6. Geometric Pattern
```css
.geometric-bg {
  background-image:
    linear-gradient(30deg, var(--color-surface) 12%, transparent 12.5%),
    linear-gradient(150deg, var(--color-surface) 12%, transparent 12.5%);
  background-size: 60px 100px;
}
```
**Effect:** Subtle repeating geometric shapes.
**When to use:** Corporate, technical, mathematical courses.

```

Card gap:           24px
Component padding:  32px
Body line-height:   1.6

Component padding:  20px

Heading line-height: 1.15
```
**Feels like:** Dashboard, reference material, data-rich. Use for technical/reference courses.

## How to Mix DNA Strands

The Art Director should:

1. **Pick a Color DNA** based on course topic and audience mood
2. **Pick a Font DNA** based on content type and cultural context
3. **Pick a Background DNA** based on color DNA (dark = glass/starfield, light = mesh/grain)
4. **Pick a Spacing DNA** based on content density
5. **Pick a Motion DNA** based on audience expectations

### Example Mix: Astronomy Course
- Color: **Dark Premium** (#0a1628 + #635bff + #00d4aa)
- Font: **Friendly Explorer** (Space Grotesk + DM Sans)
- Background: **Starfield** + **Glassmorphism** cards
- Spacing: **Comfortable** (generous negative space)
- Motion: **Energetic** (spring bouncy, longer stagger)

### Example Mix: Compliance Training
- Color: **Cool Premium** (#f8fafc + #0ea5e9 + #8b5cf6)
- Font: **Professional Trust** (Gilroy + Source Sans 3)
- Background: **Film Grain** + **Geometric Pattern**
- Spacing: **Comfortable** (clear hierarchy)
- Motion: **Subtle** (understated, professional)

### Example Mix: Arabic Leadership Course
- Color: **Warm Premium** (#1a1418 + #f59e0b + #ef4444)
- Font: **Arabic Modern** (Noto Kufi Arabic + Cairo)
- Background: **Animated Gradient** + **Film Grain**
- Spacing: **Comfortable** (RTL-optimized)
- Motion: **Subtle** (authoritative feel)
