# Playful Bright Theme

Colorful, rounded, gamified theme with warm energy and celebration.

## Best For
- Onboarding courses
- Soft skills training
- Youth/student audiences
- Courses that emphasize gamification

## Design Character
- **Mood**: Energetic, warm, encouraging
- **Typography**: Cairo (Arabic-first, rounded)
- **Palette**: Blue primary, green secondary, amber accent, warm peachy background
- **Decorations**: Floating gradient circles, dot patterns, extra-rounded corners

## Files
| File | Purpose |
|------|---------|
| tokens.json | Design token source of truth |
| theme.css | CSS custom property overrides |
| decorations.css | Floating shapes, warm gradients |
| preview.html | Visual preview of all components |
