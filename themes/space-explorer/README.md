# Space Explorer Theme

Dark cosmic theme with deep blues, glowing accents, and starfield atmosphere.

## Best For
- Science courses (astronomy, physics, space)
- Technology courses
- Any course needing a dramatic, immersive feel

## Design Character
- **Mood**: Mysterious, vast, exploratory
- **Typography**: Tajawal (Arabic-first, clean modern)
- **Palette**: Deep navy background, purple primary, cyan secondary, gold accent
- **Decorations**: Animated starfield, nebula glow orbs, card hover glow

## Files
| File | Purpose |
|------|---------|
| tokens.json | Design token source of truth |
| theme.css | CSS custom property overrides |
| decorations.css | Starfield, glow effects, nebula |
| preview.html | Visual preview of all components |

## Usage
```html
<link rel="stylesheet" href="../shared/base.css">
<link rel="stylesheet" href="../shared/theme.css">
<link rel="stylesheet" href="../shared/decorations.css">
```

## Design Quality
This theme passes the "would a designer say wow?" test through:
- Animated starfield that creates depth
- Glow effects on interactive elements
- Nebula gradients as decorative elements
- High contrast for accessibility on dark backgrounds
