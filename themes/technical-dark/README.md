# Technical Dark Theme

Code editor inspired dark theme with monospace typography and terminal aesthetics.

## Best For
- Programming/coding courses
- Technical documentation
- Developer tools training
- Computer science topics

## Design Character
- **Mood**: Precise, focused, hacker-cool
- **Typography**: JetBrains Mono for headings, Cairo for Arabic body
- **Palette**: Dark slate background, cyan primary, violet secondary, green accent
- **Decorations**: Minimal -- the sharp edges and monospace ARE the decoration

## Files
| File | Purpose |
|------|---------|
| tokens.json | Design token source of truth |
| theme.css | CSS custom property overrides |
| preview.html | Visual preview of all components |
