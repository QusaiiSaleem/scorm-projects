# Corporate Clean Theme

Light professional theme with clean lines, generous whitespace, and corporate reliability.

## Best For
- Business and compliance training
- Onboarding programs
- Financial literacy courses
- Any course needing a trustworthy, polished feel

## Design Character
- **Mood**: Professional, trustworthy, clean
- **Typography**: Noto Kufi Arabic (headings), Noto Sans Arabic (body) — crisp and readable
- **Palette**: Light background, navy primary, sky blue secondary, amber accent
- **Decorations**: None (clean design relies on whitespace and subtle shadows)

## Files
| File | Purpose |
|------|---------|
| tokens.json | Design token source of truth |
| theme.css | CSS custom property overrides |
| preview.html | Visual preview of all components |

## Usage
```html
<link rel="stylesheet" href="../shared/base.css">
<link rel="stylesheet" href="../shared/theme.css">
```

## Design Quality
This theme achieves a polished look through:
- Generous whitespace and clean typography
- Subtle, refined shadows (no heavy drop shadows)
- Professional navy-to-blue gradient hero
- Minimal decorative elements — content speaks for itself
- Tight border radius values for a crisp, corporate feel
