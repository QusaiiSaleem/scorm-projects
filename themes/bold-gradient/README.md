# Bold Gradient Theme

Vibrant gradient-rich theme with bold colors, mesh backgrounds, and energetic personality.

## Best For
- Creative and design courses
- Marketing and branding training
- Youth-oriented learning
- Any course needing energy and visual excitement

## Design Character
- **Mood**: Energetic, vibrant, modern
- **Typography**: Cairo (Arabic-first, bold and expressive)
- **Palette**: Purple-to-pink-to-orange gradient spectrum, light background
- **Decorations**: Gradient mesh background, grain texture overlay, gradient card borders

## Files
| File | Purpose |
|------|---------|
| tokens.json | Design token source of truth |
| theme.css | CSS custom property overrides |
| decorations.css | Mesh gradients, grain texture, gradient borders |
| preview.html | Visual preview of all components |

## Usage
```html
<link rel="stylesheet" href="../shared/base.css">
<link rel="stylesheet" href="../shared/theme.css">
<link rel="stylesheet" href="../shared/decorations.css">
```

## Design Quality
This theme creates visual excitement through:
- Three-color gradient spectrum (purple, pink, orange) used consistently
- Gradient mesh background for subtle depth
- Grain texture overlay for tactile feel
- Gradient borders on cards that intensify on hover
- Larger border radius values for a friendly, modern aesthetic
- Purple-tinted shadows that feel cohesive with the palette
